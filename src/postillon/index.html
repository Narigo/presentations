Was kann Blogger?

  • Kostenlos
  • Skaliert ohne Probleme
  • ... hohe Sicherheit durch Einschränkungen der Möglichkeiten
  • ... hohe Benutzerfreundlichkeit durch Minimierung der Anzahl an Features
  • ... sehr übersichtlich, da man nur in einem Template arbeiten muss
  • ... ist dank vieler Vorgaben sofort betriebsbereit
  • ... verfolgt Gamification-Ansatz: easy-to-learn, hard-to-master
  • Habe ich schon erwähnt, dass es nichts kostet?

Serverseitiges Skript:

  1. die URL auf "/s1600/" (größtmögliche Größe) anpassen
  2. herunterladen
  3. das Ergebnis davon zurückgeben
  4. Achtung: Reine Weiterleitung funktioniert nicht bei jedem sozialen Netzwerk.

... inzwischen liefert Blogger jedoch gleich größere Bilder, die auch in Facebook okay aussehen. :)
