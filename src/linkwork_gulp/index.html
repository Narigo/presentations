Gulp

Wofür Gulp?

    Automatisierung von Aufgaben

  • Im Web-Frontend Bereich z.B.
    • Compilation (CoffeScript, SASS, LESS..)
    • Minification (JS, CSS)
    • Concatenation

Warum Gulp?

  • Kleine einfache API
  • Auf Streams basierend
  • Auf maximale Nebenläufigkeit ausgerichtet
  • Code over configuration

Die Gulp API

  • gulp.task(name[, deps], fn)
  • gulp.src(globs)
  • gulp.dest(path)
  • gulp.watch(glob[, opts], tasks)
  • gulp.run(tasks...)
gulp.src| concat| uglify| gulp.dest
